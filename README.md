## Zhui

A Chinese style component library

[文档](https://zhui-team.github.io/zhui/)

可以用star来鼓励一下作者么？

## 📦 Install

```bash
npm install zhui --save
```

## 🔨 Usage

```jsx
import { Button } from 'zhui';
ReactDOM.render(<Button />, mountNode);
```

## ⌨️ Development

```bash
$ git clone https://github.com/zhui-team/zhui.git
$ cd zhui
$ npm install
$ npm run start
```
